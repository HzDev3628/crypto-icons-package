## They are even easier to use!
### Simple exampl:
``` 
import { EthIcon } from 'crypto-icons-svg'

export default function App () {
  return (
    <EthIcon className="w-10 h-10" />
  )
}
```
You can set any size for your icon. You can use <b>Tailwind</b> or <b>CSS</b>.
